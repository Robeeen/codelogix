Features of the Dynamic Header:

Top Bar Section:

1. Toggle Switch to hide or display Top Bar.
2. Email, Phone and WhatsApp Fields.
3. Option to change Font Size and Font Color.
4. Option to change height of the Top Bar.
5. Option to change Background color of the Top Bar.
6. Custom Links for Social Media Accounts.
7. Option to change Social Media Icon Color.


Main Header Section:
1. Header Background Color can be changed.
2. Header Border and Border Color can be changed.
3. Nav Menu Font Size can be changed.
4. Nav Menu Font Family can be changed.
5. Menu items space can be set.
6. Logo Can be set dynamically.
7. Logo Size, Padding can be set.
8. Toggle Switch to display or hide Call to Action Button.
9. Button text, Button background color, Text Color, Padding, Font size,
   Font family, Border can be set dynamically.

Export - Import - Reset.
1. We can export entire header settings config to a single file.
2. We can Import settings config from a JSON file.
3. We can reset the Header Design Settings completely.
